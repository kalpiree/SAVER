# Anonymity Checklist

Use this checklist before pushing the anonymous repository.

- Remove names, emails, affiliations, and acknowledgments.
- Replace absolute paths with relative paths or environment variables.
- Remove cluster-specific labels, node names, usernames, and receipts.
- Remove local caches, logs, runtime dumps, and status files.
- Check `README`, `docs/`, and script headers for identifying details.
- Check config files for local filesystem paths.
- Check tables and figures for embedded names or host labels.
- Confirm third-party code is included only if the license allows it.
- Confirm data files are public or replaced with download instructions.
- Run a final text search for names, emails, and local paths.
